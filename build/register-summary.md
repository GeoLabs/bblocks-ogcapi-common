# OGC API Common

Schemas for the OGC API - Common - Part 1: Core standard


Adds a flexible, JSON schema version agnostic documentation to the OGC API - Common - Part 1: Core standard, 
supporting further profiling by combining with other Building Blocks.

This includes:
- Landing Page (/)
- Conformance Declaration (/conformance)
- API Definition (/api)
- Common response formats (JSON, HTML)
- Common schemas (links, exceptions, etc.)
